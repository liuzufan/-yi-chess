## 这是什么？

**弈·棋道** 是一款集合了中国三大传统棋类的单文件网页游戏。不需要下载、不需要安装，打开浏览器就能玩。内置三套独立设计的 AI 引擎，每一种棋类都有专业级搜索算法支撑——五子棋能算到 18 层深度，围棋用 MCTS 蒙特卡洛树搜索（1500+ 次模拟），象棋内置 25 路开局库。棋谱回放支持 AI 实时讲解、自定义播放速度、时间记录。

> 一个 HTML 文件，三种棋类，四档难度，从入门到大师。

## 三种棋类，三种 AI

### 五子棋 — 连五为胜

15×15 棋盘，黑先落子，率先在横、竖、斜任一方向连成五子者胜。

**AI 引擎特性：**

| 技术 | 说明 |
|------|------|
| PVS 主变例搜索 | 深度 16 层，带 Aspiration Window 期望窗口 |
| VCF 连续冲四搜索 | 深度 28 层，强制对手防守的连续冲四杀棋搜索 |
| VCT 连续威胁搜索 | 深度 18 层，包含活三+冲四的复合威胁搜索 |
| 迭代加深 | 从浅到深逐层搜索，利用上一轮结果优化排序 |
| Zobrist 哈希 | 棋盘快速哈希，置换表查重，20 万条目 |
| 置换表 | 记录已搜索局面，避免重复计算 |
| 杀手着法启发 | 同层兄弟节点优先搜索曾引发剪枝的着法 |
| 历史启发 | 跨节点统计着法引发剪枝频率，优化搜索顺序 |
| 空着剪枝 | 评估"不走一步"的局面，快速剪枝 |
| 26 种开局定式 | 天元起手，涵盖浦月、花月、雨月、寒星等经典开局 |
| 七大棋型评估 | 活四、冲四、活三、眠三、活二、眠二、活一，含跳子模式 |

**四档难度对比：**

| 难度 | 搜索深度 | 时间限制 | VCF/VCT | 候选着法 | 适合 |
|------|---------|---------|---------|---------|------|
| 入门 | 6 层 | 5 秒 | 关闭 | 12 个 | 初学者 |
| 进阶 | 8 层 | 5 秒 | VCF | 12 个 | 有基础者 |
| 高手 | 10 层 | 8 秒 | VCF+VCT | 18 个 | 熟练者 |
| 大师 | 18 层 | 20 秒 | VCF(32层)+VCT(22层) | 28 个 | 挑战极限 |

### 围棋 — 围地多者胜

19 路标准棋盘，黑先落子，贴目 7.5 目。双方围地多者胜，气尽的棋子被提。

**AI 引擎特性：**

| 技术 | 说明 |
|------|------|
| MCTS 蒙特卡洛树搜索 | 1500+ 次模拟，UCT 公式平衡探索与利用 |
| RAVE 快速动作价值评估 | 利用全局着法统计加速收敛 |
| 3×3 模式匹配 | 以当前着点为中心的 9 格局部模式，匹配高频好手 |
| 征子检测 | 递归判断征子是否成立，决定提子或逃跑 |
| Benson 生死判定 | 静态识别无条件活棋，避免无意义搜索 |
| Bouzy 领地估算 | 膨胀-腐蚀法快速估算地盘归属 |
| 定式库 | 角部星位、小目、三三等常见定式变化 |
| Rollout 策略 | 提子优先、救子逃跑、局部响应、大场抢占 |

**围棋特有功能：**
- Pass 弃权手
- 数子计分（含贴目）
- 自动提子
- 劫争规则

### 象棋 — 将死为胜

标准 9×10 棋盘，红先走，将死对方将/帅者胜。

**AI 引擎特性：**

| 技术 | 说明 |
|------|------|
| 迭代加深 PVS | 深度 16 层，逐层加深 |
| 置换表 | Zobrist 哈希，快速查重 |
| MVV-LVA 排序 | 最有价值受害者-最小价值攻击者，优先搜索高收益吃子 |
| 静态搜索 | 叶子节点递归搜索吃子序列，精确评估战术交换 |
| 杀手着法 | 同层兄弟节点启发 |
| 历史启发 | 跨节点着法频率统计 |
| 空着剪枝 | depth≥3 时启用 |
| 25 路开局库 | 中炮、屏风马、反宫马、仙人指路、飞相局等 |
| 位置评估表 | 每种棋子的位置加分表，大师难度额外启用 |

**象棋开局库涵盖：**
> 中炮对屏风马 · 中炮对屏风马平炮兑车 · 仙人指路 · 仙人指路对卒底炮 · 飞相局 · 飞相局对左中炮 · 仕角炮 · 反宫马 · 中炮横车七路马 · 顺炮直车对横车 · 列手炮 · 中炮巡河炮 · 五七炮对屏风马 · 五八炮对屏风马 · 中炮七路马对屏风马 · 中炮巡河车对屏风马 · ...共 25 路

## 游戏功能

### 核心玩法
- **先手选择** — 每局可选玩家先手或 AI 先手
- **实时胜率显示** — AI 思考时实时显示当前局面评估
- **AI 思考标签** — 显示 AI 当前使用的策略（开局定式 / VCF 搜索 / 威胁搜索 等）
- **悔棋功能** — 五子棋和象棋支持撤回上一步

### 棋谱系统
- **自动保存** — 每局对弈自动保存到浏览器本地存储，记录每步耗时
- **自定义命名** — 棋谱列表中可随时重命名对局记录
- **时间显示** — 回放时显示每步耗时和总时长
- **回放查看** — 逐步回放整局棋路，支持上一步/下一步/自动播放
- **自定义播放速度** — 0.5x / 1x / 2x / 4x 四档速度自由切换
- **AI 实时讲解** — 回放每一步时自动生成 AI 讲解（棋型分析、战术点评、吃子价值评估）
- **下载 HTML** — 将棋谱导出为独立 HTML 文件，双击即可打开回放，可分享给好友

### 视觉设计
- **中式美学** — 宣纸质感棋盘、木质纹理、毛笔字风格棋子
- **程序化纹理** — 木纹纹理用种子里程化生成，无图片依赖
- **棋子渲染** — 多层径向渐变 + 高光 + 投影 + 边缘反射，模拟玉石质感
- **落子动画** — 最后一手高亮标记
- **响应式布局** — 适配桌面和移动端屏幕

## 玩法指南

### 五子棋怎么玩？

1. 进入主菜单，点击「五子棋」
2. 选择难度（建议从「进阶」开始）
3. 选择先手（黑先或白先）
4. 点击棋盘交叉点落子
5. 在横、竖、斜方向率先连成 5 子获胜

**技巧：**
- 大师难度下 AI 会使用 26 种开局定式，你能学到经典开局
- 活三必须防，活四已无法防
- 注意跳子模式，跳三跳四同样致命

### 围棋怎么玩？

1. 进入主菜单，点击「围棋」
2. 选择难度
3. 选择先手
4. 点击棋盘交叉点落子
5. 围地多者胜，黑棋贴目 7.5 目

**技巧：**
- 角部价值最高，其次是边，最后是中腹
- AI 会使用定式，你可以学习角部常见变化
- 注意气的概念，气尽的棋子会被自动提掉
- 不想继续可以点「Pass」弃权

### 象棋怎么玩？

1. 进入主菜单，点击「象棋」
2. 选择难度
3. 选择先手（红先或黑先）
4. 点击棋子选中，再点击目标位置移动
5. 将死对方将/帅获胜

**技巧：**
- 大师难度下 AI 使用 25 路开局库，能应对主流开局
- 车马炮价值最高（900/400/450），兵卒价值最低（100）
- 选中棋子后会显示黄色圆圈标记
- 注意飞将规则（将帅不能在同一直线无遮挡相对）

## 技术架构

### 技术栈

- **纯原生 HTML / CSS / JavaScript** — 零框架依赖，单文件 2500+ 行
- **Canvas 2D** — 棋盘和棋子全部用 Canvas 程序化绘制
- **Web Storage API** — localStorage 存储棋谱记录
- **异步 AI 计算** — setTimeout 分块执行，防止主线程阻塞

### AI 算法

| 棋类 | 核心算法 | 搜索深度 | 特色 |
|------|---------|---------|------|
| 五子棋 | PVS + VCF/VCT | 18 层 | 威胁搜索(32/22层) + 26 开局定式 |
| 围棋 | MCTS + RAVE | 1500+ 模拟 | 征子检测 + 3×3 模式匹配 |
| 象棋 | PVS + 静态搜索 | 16 层 | MVV-LVA + 25 路开局库 |

### 代码结构

## 常见问题

**Q: AI 思考时页面会卡吗？**
A: 不会。所有 AI 计算都使用异步分块执行（setTimeout），主线程保持响应，你可以随时点击退出。

**Q: 棋谱会丢失吗？**
A: 棋谱保存在浏览器 localStorage 中，只要不清除浏览器数据就不会丢失。清除浏览器数据后棋谱会消失。

**Q: 下载的 HTML 棋谱怎么用？**
A: 双击下载的 `.html` 文件，用浏览器打开即可查看回放。文件是完全独立的，可以分享给任何人。

**Q: 手机能玩吗？**
A: 能。游戏使用响应式设计，手机浏览器打开即可触摸操作。

**Q: 需要联网吗？**
A: 不需要。下载 `弈.html` 后完全离线可玩，AI 全部在本地计算。
